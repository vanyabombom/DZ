What is HTML?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the documentâ€™s appearance.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive forms can be embedded. HTML provides structured documents using tags that denote headings, paragraphs, lists, and other elements. HTML tags are not displayed by browsers but are used to interpret page content.

Markup

HTML markup includes tags and attributes, character-based data types, and references. Tags usually come in pairs like <h1> and </h1>, though some, like <img>, are unpaired and called empty elements.

Another essential component is the HTML document type declaration, which triggers standards mode rendering.

Example of a "Hello, World!" program:

            
<!DOCTYPE html>
<html>
    <head>
        <title>This is a title</title>
    </head>
    <body>
        <p>Hello world!</p>
    </body>
</html>
